# Utilities
You can use a utility on any element to modifiy text, element placement

## Text Sizes

| Class        | Default value | Description    
| ------------- |:-------------:
| `.u-text-xxs` | 1rem      | Modify the text size to xxs
| `.u-text-xs` | 1.2rem      | Modify the text size to xs
| `.u-text-s` | 1.5rem | Modify the text size to s
| `.u-text-m` | 1.8rem | Modify the text size to m
| `.u-text-l` | 2.4rem | Modify the text size to l
| `.u-text-xl` | 3rem | Modify the text size to xl
| `.u-text-xxl` | 3.6rem | Modify the text size to xxl
| `.u-text-xxxl` | 4.8rem | Modify the text size to xxxl

## Other Text Properties

| Class        | Description    
| ------------- |:-------------:
| `.u-text-normal`      | Normal font weight for text
| `.u-text-strong`      | Strong font weight for text
| `.u-text-center` | Align text to center
| `.u-text-nowrap` | Nowrap text
| `.u-text-one-liner` | Put a text in only one line without a break and add dots to the end
| `.u-text-uppercase` | Uppercase text
| `.u-line-height-reset` | Reset the line-height to 1


## Aspect Ratios

| Class        | Description    
| ------------- |:-------------:
| `.u-aspect-ratio-16x9`      | Create an element with aspect ratio of 16x9
| `.u-aspect-ratio-21x9`      | Create an element with aspect ratio of 21x9
| `.u-aspect-ratio-1x1` | Create an element with aspect ratio of 1x1
| `.u-aspect-ratio-4x3` | Create an element with aspect ratio of 4x3
