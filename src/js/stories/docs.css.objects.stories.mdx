## Distances
There are a lot of classes to handle distances between items. **Distances used margin-top.**

| Class         | Default value | Description    
| ------------- |:-------------:
| `.o-distance` | 32px | Standard space. Mostly use between rows or sections.
| `.o-distance-xs` | 4px | Create the smallest space.
| `.o-distance-s` | 8px | Create a small space.
| `.o-distance-m` | 16px | Create a medium space
| `.o-distance-l` | 32px | Create a large space
| `.o-distance-xl` | 64px | Create a x-large space

## Flexbox
There are a lot of classes to handle distances between items. **Distances used margin-top.**

| Class         | Description    
| ------------- |:-------------:
| `.o-flex` | Put items in a row
| `.o-flex--end` | Align items and justify content to the end
| `.o-flex--center` | Align items to center
| `.o-flex--justify-center` | Justify content to center
| `.o-flex--justified` | Justify items to space-between
| `.o-flex--large-justified` | Justify item to space-between from large breakpoint to up

## Prevent space
Prevent elements to move on the left or right edge of the browser window.

| Class         | Default value | Description    
| ------------- |:-------------:
| `.o-prevent-space` | 16px | Prevent items to be on the left and right edge of the browser.
