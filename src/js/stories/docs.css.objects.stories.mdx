## List Inline
Show items in a line

    <div>
        <ul className='o-list-inline'>
            <li className='o-list-inline__item'>Item</li>
            <li className='o-list-inline__item'>Item</li>
            <li className='o-list-inline__item'>Item</li>
        </ul>
    </div>

## List Bare
Show items among themselves

    <div>
        <ul className='o-list-bare'>
              <li className='o-list-bare__item'>Item</li>
              <li className='o-list-bare__item'>Item</li>
              <li className='o-list-bare__item'>Item</li>
        </ul>
    </div>
