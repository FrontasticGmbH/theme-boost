# Theming

<img src={imgTheming} />

**Don't**
````scss
.c-header {
  background: $color-blue;
  color: $color-white;
}
````
````html
<header role="banner" class="c-header">
  <h1>Title</h1>
</header>
````

**Do**
````scss
// _components.header.scss
.c-header {
  padding: 20px;
}

// _themes.spotlight.scss
.t-spotlight {
  background: $color-blue;
  color: $color-white;
}
````
````html
<header role="banner" class="c-header t-spotlight">
  <h1>Title</h1>
</header>
````

## Available Themes

| Class        | Description    
| ------------- |:-------------:
| `.t-quietlight`      | Create a quiet background and **don't** put an element in focus
| `.t-quietlight__input`      | Create a quiet input
| `.t-spotlight` | Create a spot background and **put the element in focus.**
| `.t-spotlight-border` | Create a spot border
| `.t-spotlight-color` | Create a spot text.
| `.t-universe` | Create a real dark background

## Text

| Class        | Description    
| ------------- |:-------------:
| `.t-text-quiet`      | Create a quiet text color
| `.t-text-ghost`      | Create a text with one of the lightest color in this project
| `.t-text-danger` | A danger text like error messages
| `.t-text-focus` | A focus text with a lot of attention
| `.t-text-uniform` | Create a normal text color like the body text
| `.t-text-spotlight` | A spot text with attention
