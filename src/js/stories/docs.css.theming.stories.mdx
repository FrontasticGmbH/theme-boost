# Theming

<img src={imgTheming} />

**Don't**
````scss
.c-header {
  background: $color-blue;
  color: $color-white;
}
````
````html
<header role="banner" class="c-header">
  <h1>Title</h1>
</header>
````

**Do**
````scss
// _components.header.scss
.c-header {
  padding: 20px;
}

// _themes.spotlight.scss
.t-spotlight {
  background: $color-blue;
  color: $color-white;
}
````
````html
<header role="banner" class="c-header t-spotlight">
  <h1>Title</h1>
</header>
````
